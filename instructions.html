Custom Stats Tracking Template - Info

Purpose:

My goal with template is to provide a starting resource allowing teams to track custom statistics. There are a lot of opportunities within the framework for customization, allowing teams to build a play-by-play data set for internal statistics.

Usage guide:

  • Within the code files, one can change the variables, customize the play calls or alter the set-up to track defensive statistics.
  • Users can choose to track some or all the metrics in a given session. Leave the variable at "-" if you wish to not track the variable.
  • Once the possession is complete, press the "Submit" button to log the data in the table below.
  • The default table will display up to 100 possessions, and can be sorted by any of the columns.
  • Once the session is complete, click on the "Export CSV" button to download the data for analysis.
  • Note: This template is not linked to a database, so refreshing the url will cause data to be lost. Liking the form to a database is possible

    • Variable Definitions:

    • PG: Point Guard who is currently in the game - Can be changed to "Combo/Swing/Wing"
    • SG: Shooting Guard who is currently in the game - Can be changed to "Combo/Swing/Wing"
    • SF: Small Forward who is currently in the game - Can be changed to "Swing/Wing"
    • PF: Power Forward who is currently in the game - Can be changed to "Swing/Wing"
    • C: Center who is currently in the game - Can be changed to "Bigs"

    • Possession Start (Pos Start): The type of situation that initiated the possession
    • Play Call: Custom inputs to track the play names for a team
    • Play Movement: Generalized description of the path taken by the ball during the possession
    • Passes: Number of passes (excluding inbound) during the possession
    • Shot Selection: Type of shot attempt
    • Shot Quality: Indicates if the shot attempted was open or contested
    • Play Result: Result of the actions executed during the possession
    • Player: Player who attempts the shot or turns the ball over
    • Opponent: Oponnent for the events
    • Arena: Home / Away / Neutral site game
